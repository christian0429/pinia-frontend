계산된 속성 (Computed Property):
-------------------------------

특징:
Vue 컴포넌트 내에 선언된 데이터 속성들을 기반으로 계산되는 속성입니다.
데이터의 변화를 감지하여 자동으로 업데이트됩니다.
캐싱을 사용하여 연산 결과를 저장하므로 동일한 입력에 대해 여러 번 호출되어도 계산이 한 번만 수행됩니다.
활용:
템플릿에서 반복적으로 사용되는 데이터의 연산이나 필터링에 유용합니다.
복잡한 데이터 변환 또는 조작이 필요한 경우 사용됩니다.


메서드 속성 (Methods):
-------------------------------

특징:
Vue 컴포넌트 내에 선언된 함수 형태의 속성입니다.
필요할 때마다 명시적으로 호출되어 실행됩니다.
데이터의 변화를 자동으로 감지하지 않으며, 수동으로 호출되어야 합니다.
활용:
사용자 이벤트나 조건에 따라 동적으로 처리해야 하는 로직에 사용됩니다.
특정 동작을 트리거할 때 사용됩니다.


관찰 속성 (Watch Property):
-------------------------------
특징:
Vue 컴포넌트 내에 선언된 데이터의 변경을 감시하고 이에 반응하는 속성입니다.
데이터의 변화를 추적하고, 변화가 감지되면 설정된 콜백 함수가 실행됩니다.
비동기 작업이 필요한 경우 사용될 수 있습니다.
활용:
특정 데이터의 변화에 대한 트리거가 필요한 경우 사용됩니다.
데이터의 변화에 따라 비동기적으로 다른 작업을 수행해야 하는 경우 사용됩니다.
수정 전/후 데이터를 이용해 특정한 작업을 수행할 때 사용 가능합니다.



이러한 속성들은 각자의 특성에 따라 다른 상황에서 유용하게 사용될 수 있으며,
Vue 애플리케이션에서 데이터 처리와 상태 관리를 유연하게 구현하는 데 도움이 됩니다.
